# ROC DZ вЂ” Laptop Store Algeria

## Run & Operate

- `pnpm --filter @workspace/api-server run dev` вЂ” run the API server (port 8080)
- `pnpm run typecheck` вЂ” full typecheck across all packages
- Required env secrets: `MONGODB_URI`, `CLOUDINARY_CLOUD_NAME`, `CLOUDINARY_API_KEY`, `CLOUDINARY_API_SECRET`

## Stack

- pnpm workspaces, Node.js 24, TypeScript 5.9
- API: Express 5
- DB: **MongoDB + Mongoose** (Atlas cloud)
- Images: **Cloudinary** (upload via `/api/upload`, returns Cloudinary URL directly)
- Validation: Zod (`zod/v4`)
- API codegen: Orval (from OpenAPI spec) вЂ” `@workspace/api-zod`, `@workspace/api-client-react`
- Build: esbuild (ESM bundle)
- Frontend: React + Vite + Wouter + TailwindCSS (dark purple gaming aesthetic)

## Where things live

- `artifacts/api-server/src/models/` вЂ” Mongoose models (laptop, accessory, category, order, promo-code)
- `artifacts/api-server/src/lib/mongodb.ts` вЂ” MongoDB connection
- `artifacts/api-server/src/routes/` вЂ” Express route handlers
- `artifacts/api-server/src/routes/upload.ts` вЂ” Cloudinary upload endpoint
- `artifacts/roc-dz/src/` вЂ” React frontend
- `artifacts/roc-dz/src/context/CartContext.tsx` вЂ” Cart state (IDs are strings/MongoDB ObjectIds)
- `lib/api-zod/` вЂ” Zod schemas (generated by Orval from OpenAPI spec)
- `lib/api-client-react/` вЂ” React Query hooks (generated by Orval)

## Architecture decisions

- MongoDB ObjectIds used as `id` (string) everywhere вЂ” Mongoose `toJSON` transform maps `_id` в†’ `id`, removes `__v`
- Cart items use `laptopId: string` (was `number` with 100000-offset hack for accessories вЂ” now clean string IDs + `isLaptop` flag)
- Cloudinary upload: single POST to `/api/upload` (multipart), returns `{ url }` вЂ” no more 2-step presigned URL flow
- `lib/db` (Drizzle/PostgreSQL) is no longer used by api-server вЂ” models are local to api-server
- Express 5 wildcard routes require `/{*path}` syntax (not bare `*`)

## Product

- Admin panel: manage laptops, accessories, categories, orders, promo codes, media uploads

## User preferences

- French-language UI for Algerian market

## Gotchas

- MongoDB Atlas **Network Access** must allow `0.0.0.0/0` (all IPs) for Replit + Vercel to connect
- `MONGODB_URI` is a secret (not env var) вЂ” do not use `setEnvVars` to update it
- Cloudinary credentials: cloud name is env var, API key + secret are secrets
- `@workspace/db` (Drizzle) is still present as a workspace package but NOT imported by api-server
- esbuild bundles mongoose вЂ” `mongodb-client-encryption` is externalized in `build.mjs`
